How to Create Custom Booking Codes

Custom Booking Codes Creating Custom Codes - SportyBet allows creation of custom booking codes for sharing with other users. Steps to Create a Custom Booking Code: 1. Go to the Load Option section via menu options > Others. 2. Click the green icon at the top right. 3. Enter a booking code in the field and click Assign to generate a new custom code. 4. Edit the booking code as needed. Note: Custom Code Name can be up to 25 characters and only contain lowercase letters or numbers.
