How to Edit a Bet

 Overview Edit Bet is a feature that allows you to modify unsettled bets by adding or removing selections and increasing the stake. This feature is available for both Pre-match and Live bets. Supported Bet Types - Edit Bet is available for Single and Multiple bets. - Edit Bet is not supported for System Bets, Flexi Bets, or One-Cut Bets. - Bets placed using Sporty Coins or Gifts are not eligible for Edit Bet. Eligibility Criteria - Edit Bet is offered only when Cashout is available for all unsettled selections in your bet. - The outcome of the bet must not have been determined. - If some selections in a Multiple Bet are already settled, Edit Bet can still be used for the remaining unsettled selections. Important Notes - When you edit a bet, all prices (odds) on the new bet will be updated to reflect the current odds, not the original odds. How to Use Edit Bet 1. Place a bet. 2. Open the Open Bets tab. 3. Locate the bet you wish to edit. 4. Use the options to: - Add a selection. - Remove any of the selections. - Increase the stake. 5. If you remove a selection, you can undo the removal by clicking the Undo button next to the selection. 6. Click the Save Changes button to confirm your edits. 7. After saving changes, a Cashout of the original bet will be conducted, and the new modified bet will be placed. Note If you have tried the above steps and are still unable to edit your bet, the case should be handed off to a human agent.
