How to Use a Booking Code to Place a Bet

How to Use the Booking Code Overview This section explains how to use a booking code to place a bet. Steps to Use a Booking Code 1. Log in to your account. 2. Go to the betslip. 3. If you have any existing selections, click on Remove All to clear them. 4. Locate the empty field provided for booking codes. 5. Enter the booking code you have received (for example, from friends). 6. Proceed to place your bet. Note Booking codes from any region can now be used without switching regions as was previously required.
