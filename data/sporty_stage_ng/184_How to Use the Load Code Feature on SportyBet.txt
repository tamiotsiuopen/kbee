How to Use the Load Code Feature on SportyBet

What is Load Code? Overview The Load Code feature allows users to load one or multiple booking codes that have been shared by another user. How to Use the Load Code Feature 1. Go to the A-Z menu on SportyBet. 2. Click on Load Code. 3. Enter the booking code and load it. 4. To load another code, return to the Load Code option and enter the next booking code. 5. Repeat this process to load as many booking codes as you want, one after the other. Limits - You can add up to 50 selections for Single and Multiple bets. - You can add up to 15 selections for System bets.
