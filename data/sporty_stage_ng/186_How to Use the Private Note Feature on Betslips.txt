How to Use the Private Note Feature on Betslips

Private Note Feature Overview The Add Private Note feature allows you to write a personal note for your bet immediately after placing it. How to Add a Private Note 1. Place your bet. 2. On the Bet Successful screen, click on Add Private Note. 3. Write your personal note for the bet. Where the Note Appears - The note will be displayed below the betslip on your Bet History page. Editing Your Note - You can edit the note later from the betslip in your Bet History. Platform Availability - This feature is currently available on Android and Mobile Browser platforms. Important Note - You can only add a private note immediately after placing the bet. Once you leave the Bet Successful screen, the option to add a note is no longer available.
