How to Check Results of Completed Events

Check Result of an Event That Has Ended Overview SportyBet provides information about the results of completed events. You can check the outcome of any event that has ended by following the steps below. How to Check Event Results 1. Click on the A-Z Menu located at the bottom of the page. 2. Select Results from the Menu Bar. 3. Enter the 4 or 5-digit Game ID in the search bar. 4. All results for that event will be displayed. Note If you have followed these steps and still cannot find the result, the case should be handed off to a human agent.
