Troubleshooting: Unable to Play Sporty Soccer

Unable to play Sporty Soccer Overview This guide addresses issues related to being unable to play Sporty Soccer, including incomplete payable attempts, device changes, and session conflicts. Incomplete Payable Attempts - If a payable attempt is started but not concluded (for example, if the app is exited, the phone is turned off, or the battery dies), the customer has 14 days to complete the attempt. - After 14 days, any incomplete bet will be considered Lost. Device Change Restrictions - If a customer changes devices between the first and second attempt, the first session must be concluded on the previous device before continuing on the new device. Multiple Device Login Prompt - If a prompt appears indicating multiple devices have been used to log in: - Log out sessions from other devices, or - Reset the password using the OTP option. - To reset your password, use the provided password reset option. Unable to Conclude Session on Previous Device - If the customer cannot conclude the first session on the previous device, the previous game session can be terminated by the SportyBet team. - Termination of the previous game session will mark the round as Lost. Note If the above solutions have been tried and did not work, the case should be handed off to a human agent.
