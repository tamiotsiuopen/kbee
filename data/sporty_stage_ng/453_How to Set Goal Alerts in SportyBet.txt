How to Set Goal Alerts in SportyBet

 Overview SportyBet provides a feature that allows you to receive pop-up notifications whenever your selected team scores a goal. This helps you stay updated even when you are not actively using the SportyBet app. Note: Pop-up notifications will only appear if you are not using the SportyBet app at the time the goal is scored. Setting a Goal Alert for a Match 1. Open the SportyBet app. 2. Select the match for which you want to receive goal alerts. 3. On the match screen, tap the bell icon to add the match to your alert list. Enabling Match Alerts in Notification Settings To ensure you receive goal alerts, you must enable Match Alerts in your notification settings: 1. Go to the Home Page of the app. 2. Tap on 'ME' (your profile section). 3. Tap the Settings icon in the top right corner. 4. Select 'Notification Settings.' 5. Tap on 'Match Alerts.' 6. Toggle 'Receive Match Alerts' to ON. Alert Options - You can choose to receive alerts for all matches you have added. - Alternatively, you can select specific matches from your list to receive notifications only for those matches.
