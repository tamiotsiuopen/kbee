Understanding Flexi Bets: How They Work and How to Place Them

🔵 What is a Flexi Bet? A Flexi Bet is a type of Multiple Bet that allows you to win even if not all your selections are correct. Unlike a traditional multiple bet, where every selection must win, a Flexi Bet lets you set a minimum number of winning selections (the Flexi number) for a payout. ✅ Benefit: Reduces risk—your entire bet isn’t lost if some selections fail. ✅ Availability: Works on pre-match and live betting. ✅ Cash Out: Eligible for Cash Out. 🔵 How Flexi Bets Work Select multiple matches (up to 50 selections). Set a minimum number of selections required for a payout (Flexi number). If correct selections ≥ Flexi number, you win. If correct selections < Flexi number, the ticket is lost. If all selections win, payout is calculated automatically. 🟢 Example Scenarios Scenario 1: Selections: 7 matches Flexi minimum: 3 Outcome: 5 selections win ✅ Result: You win, because you met the Flexi requirement. Scenario 2: Selections: 20 matches Flexi minimum: 15 Outcome: 14 selections win ❌ Result: You lose, as you did not meet the minimum. Scenario 3: Selections: 20 matches Flexi minimum: 15 Outcome: All 20 selections win ✅ Result: You win; payout calculated for all winning selections. 🔵 How to Place a Flexi Bet Make multiple selections (1 per match, up to 50). Below the stake entry, go to “Insure” section and select Flexi. Set “Options to be Correct” to define minimum winning selections. 🟠 Special Rules and Conditions Ticket wins if correct selections ≥ Flexi minimum. Voided selections: If the required number of winning selections is still met → full winnings credited. If required selections aren’t met but voided selections would have met it → payout recalculated (may be less than stake). Certain multiple bets limit Flexi options. Example: 20 selections may require minimum 14. 🟠 Important Notes Flexi Bet Example: Flexi 3 out of 7, 5 win → ✅ Win Fewer than 3 win → ❌ Lose Platform can revoke winnings if fraudulent activity is detected. Unresolved issues → escalated to human agent.
